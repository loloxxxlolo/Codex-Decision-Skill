# Example: Memory Write-Back

## User Message

Ok, let's record this: I will not build the cloud sync layer yet. The mobile app and desktop memory should run independently first.

## Correct Memory Update

Add to the relevant project:

```md
## summary

- The mobile life-recording app and desktop decision-memory system should run independently for now.
- A cloud aggregation layer should be considered only after both loops prove useful.

## warning

- Do not build a unified cloud AI assistant too early.

## log

- YYYY-MM-DD: Confirmed that mobile and desktop memory loops should stay decoupled for now.
```

## Incorrect Memory Update

Do not invent:

- a cloud architecture,
- API design,
- sync protocol,
- user account system,
- product roadmap,
- migration plan.

Those are AI suggestions unless the user explicitly confirms them.
